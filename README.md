# Mortgage Repayment Calculator
A Mortgage Repayment Calculator is a digital tool designed to estimate how much your monthly mortgage payment will be. It helps you understand the long-term financial commitment of buying a home by breaking down the costs based on a few key variables.

## Overview
A mortgage repayment calculator built with React, focusing on accessibility and responsiveness.

## Features
* Input mortgage information and display monthly repayment and total repayment amounts
* Form validation and accessibility features
* Responsive design for different screen sizes

## Technologies Used
* React
* React Hooks (useState, useEffect)
